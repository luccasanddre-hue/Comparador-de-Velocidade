# 🚗 Comparador de Velocidade de Veículos

Projeto desenvolvido utilizando HTML, CSS e JavaScript.

## 📋 Descrição

O usuário informa o nome e a velocidade de dois veículos. O sistema compara as velocidades e informa qual veículo é o mais rápido ou se ambos possuem a mesma velocidade.

## ✨ Funcionalidades

- Comparação entre dois veículos
- Validação dos campos
- Interface responsiva
- Exibição dinâmica do resultado

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 📚 Conceitos praticados

- Manipulação do DOM
- Eventos
- Condicionais (`if`, `else if`, `else`)
- Validação de dados
- Template Literals
